# Frontend для Event Sourcing CRM

## Описание

Проект находится в стадии разработки. Полноценного взаимодействия с сервером еще нет, но уже готовы большая часть
компонентов. Для лидов, клиентов и других сущностей созданы страницы с моковыми данными для наглядности.

## Планируется добавить

1. Взаимодействие с сервером
2. Адаптивная верстка
